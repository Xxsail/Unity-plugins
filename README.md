## Overview

This repository contains a curated collection of Unity plugins designed to streamline game development. Each plugin is modular and can be integrated independently into your Unity projects.

## Features

- **Modular Design**: Each plugin can be used independently
- **Easy Integration**: Simple installation process
- **Well Documented**: Comprehensive documentation for each plugin
- **Performance Optimized**: Built with performance in mind
- **Unity Package Manager Support**: Easy installation via UPM

## Requirements

- Unity 2020.3 LTS or higher (recommended)
- Compatible with Unity 2019.4 LTS and above
- Supports all major platforms (Windows, macOS, Linux, iOS, Android, WebGL)

## Installation

### Method 1: Unity Package Manager (Git URL)

1. Open Unity Package Manager (`Window > Package Manager`)
2. Click the `+` button in the top-left corner
3. Select `Add package from git URL...`
4. Enter the repository URL: `https://github.com/Xxsail/Unity-plugins.git`
5. Click `Add`

### Method 2: Manual Installation

1. Download the latest release from the [Releases](https://github.com/Xxsail/Unity-plugins/releases) page
2. Extract the package
3. Copy the desired plugin folders into your Unity project's `Assets/Plugins` directory

### Method 3: Clone Repository

```bash
cd YourUnityProject/Assets
git clone https://github.com/Xxsail/Unity-plugins.git Plugins
```
